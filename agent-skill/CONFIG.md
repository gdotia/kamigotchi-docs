# Kamigotchi Configuration

## Play Mode
- `autonomous` — Agent plays automatically (default)
- `interactive` — Agent asks before each action

## Strategy
- `harvester` — Focus on resource gathering
- `quester` — Prioritize quest completion
- `trader` — Focus on profitable trades
- `balanced` — Mix of all activities (default)

## Settings
- `auto_level`: true — Auto level-up when XP sufficient
- `auto_equip`: true — Auto equip better gear
- `harvest_collect_interval`: 30m — How often to collect harvests
- `notification_level`: summary — summary | detailed | silent
